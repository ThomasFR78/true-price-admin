Mandat 1/3 : "True Price Authority" (Core Project)
Objet : Création du Comparateur et de son Média
Deadline MVP : 15 Février 2026

La Mission (The "Why")
Auditer la Vérité Terrain pour devenir la "Source de Vérité" (Trust Authority).
Philosophie : "Startup Studio" (Projet Externe, Standalone). Tu as carte blanche.

Le Cahier des Charges Détaillé
A. Le Moteur "True Price" (Human-First Watch)
Objectif : Calculer le "Taux de Mensonge" et décerner les bons/mauvais points.
Scope Pilote : 3 Jeux Majeurs (CD Key) + 1 Hardware (POC Carte Graphique).

#### 1. Audit des Comparateurs (AKS vs GG.deals)
Qui affiche le meilleur prix ? Qui ment sur les frais ?

#### 2. Audit des Marchands (Vendor Integrity - NOUVEAU)
Nous ne jugeons pas que les comparateurs. Nous jugeons les Vendeurs (Eneba, Kinguin, HRK, Instant Gaming).
Fake Promos : Le prix barré est-il réel ou gonflé ?
Frais Cachés : Ajoutent-ils une "Protection Acheteur" obligatoire cachée ?
Output : Un classement "Most Honest Vendor" vs "Scam Alert".

#### 3. Audit des Coupons (Le Crash-Test)
Process : Test manuel de validité ("AKS10" fonctionne-t-il vraiment ?).
Métriques : Validité (Oui/Non) et Valeur Réelle.

B. La "Media Factory" (L'Usine à Contenu IA)
CRITIQUE : La donnée ne vaut rien si elle n'est pas publiée.
Infographies Auto : "Le Baromètre du Mensonge" (Hebdo pour Discover).
Vidéos Shorts : "On a testé 5 vendeurs, voici le résultat".
Reviews Data-Driven : "Avis Eneba 2026 : Arnaque ou Bon plan ? Nos 50 tests mystères".

C. "The AI Oracle" (Stratégie MCP)
Concept : Ne pas remplacer les comparateurs, mais guider l'IA.
Livrable : Un endpoint MCP qui renvoie le "Trust Score" et le "Best Price Verified".

D. Le "Cashback Comparator" (Monétisation)
Mission : Comparer les offres de Cashback (iGraal vs Direct).
Output : Intégration simple ("Si vous achetez ici, vous récupérez X€").

Tes Moyens
Tech : 100% Libre (No-Code, Python, etc.).
Budget Ops : Staffing Offshore (VA) + APIs.

Livrables Attendus (15 Mai)
Dashboard Public : Classement "Verified Trust" (Comparateurs + Marchands).
Média Actif : Des contenus (Infographies/Vidéos) publiés régulièrement.
API MCP : Fonctionnelle.
Rapport Cashback : Étude de rentabilité.
